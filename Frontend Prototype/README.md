# Frontend Prototype

This folder contains the frontend prototype demonstration for the **AI Resume Analyzer** product.

The prototype illustrates how users interact with the system to upload their resume and receive AI-generated analysis and feedback.

* Resume upload interface
* User interaction flow
* Resume analysis output
* AI-generated feedback dashboard

The video demonstrates the user journey from **resume submission to AI-powered evaluation and feedback generation**.
